# Amazon-Sales-Dataset-Analysis
This repository contains a Jupyter notebook exploring Amazon sales data, focusing on analysis, visualizations, and insights into sales trends across product categories and over time.

## Objective
- Perform exploratory data analysis (EDA) on Amazon sales data.
- Visualize sales trends over time, across categories, and by geography.
- Identify seasonal patterns, top-performing products, and key sales influences.
